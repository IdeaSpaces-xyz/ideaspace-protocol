# Purpose

Give knowledge repos a **fixed, platform-neutral shape** so that any conformant agent or tool can inhabit any conformant space — orient, navigate, and work — with no bespoke instructions.

The shape is the standard: Markdown knowledge under git, how-to-work in `_agent/`, identity and history in git, the same building block at every scale. Predictability is the point — drop a conformant agent into a conformant space and it already knows where to look.

This repo keeps the spec, the machine-readable schema, the reference implementation, and the conformance kit **together**, so the definition and a working implementation never drift apart. Other tools and runtimes build on it.
